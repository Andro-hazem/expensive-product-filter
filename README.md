# Expensive Product Filter

A Python automation script that fetches product data from an API, filters expensive products, and saves the results as CSV or JSON. Includes logging and supports command-line arguments for flexible use.

## Features

- Fetch products from any API URL
- Filter products above a specified price
- Save results as CSV and JSON
- Logs actions with timestamps
- Command-line arguments for easy configuration

## Installation

1. Clone this repository:
```bash
git clone https://github.com/Andro-hazem/expensive-product-filter.git
```
## Dependencies

- Python 3.10+  
- requests
